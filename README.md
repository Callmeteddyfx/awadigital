# awadigital
This repository hosts the source code for my freelance portfolio website, **AwA Web Studio**.  
The site showcases my skills in **web development, graphic design, and digital content creation**, and demonstrates real or demo projects to potential clients.

## Features
- Modern, responsive design built with HTML, CSS, and React
- Portfolio showcase with project descriptions
- Contact form for inquiries
- Optimized for performance and SEO
- Hosted on GitHub Pages with a custom domain via Cloudflare

## Purpose
This portfolio is designed to establish credibility as a freelance web developer, provide examples of my work, and help potential clients understand the services I offer.

## Tech Stack
- HTML, CSS, JavaScript
- React (for dynamic components)
- GitHub Pages for hosting
- Cloudflare for DNS and SSL
